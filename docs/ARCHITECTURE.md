## Security Analysis

### Threat Model

| Threat | Traditional | ZKP Solution |
|--------|-------------|--------------|
| Network eavesdropping | Password leaked | Only proof values (useless to attacker) |
| Server compromise | Password hashes leaked | Only public values (can't derive password) |
| Replay attack | Possible | Prevented (fresh challenge each time) |
| Man-in-the-middle | Password captured | Only meaningless proof values captured |

### Cryptographic Hardness

The security relies on the **Discrete Logarithm Problem (DLP)**:

```
Given: y = g^x mod p
Find:  x

This is computationally infeasible for large p (256+ bits)
Estimated time to solve: > 10^50 years (current technology)
```
